Kafka uses key-value pairs in the [[http://en.wikipedia.org/wiki/.properties][property file format]] for configuration. These values can be supplied either from a file or programmatically.

* Broker Configs
The essential configurations are the following:

- =broker.id=
- =log.dirs=
- =zookeeper.connect=
